# SecureChat Monorepo

Монорепозиторий для SecureChat с использованием pnpm workspaces.

## Структура

- `apps/`
  - `server/` — серверное приложение (каркас).
  - `web/` — веб‑клиент (каркас, планируется перенос из `legasy`).
  - `mobile/` — мобильное приложение на Expo.
- `packages/`
  - `protocol/` — общие типы и протоколы обмена.
  - `db/` — работа с базой данных.
  - `crypto/` — криптографические примитивы и утилиты.
  - `storage/` — абстракции над хранилищами (IndexedDB и др.).
- `legasy/` — существующее веб‑приложение, источник для будущей миграции в `apps/web` и `packages/*`.

## Основные команды

- `pnpm install` — установка зависимостей монорепы.
- `pnpm dev:web` — запуск веб‑клиента.
- `pnpm dev:server` — запуск сервера.
- `pnpm dev:mobile` — запуск мобильного клиента.
- `pnpm dev:legacy` — запуск текущего легаси‑фронтенда из каталога `legasy`.
- `pnpm build:web` — сборка веб‑клиента.
- `pnpm build:server` — сборка сервера.
- `pnpm build:mobile` — сборка мобильного клиента.
- `pnpm lint` — запуск линтера во всех пакетах/приложениях.
